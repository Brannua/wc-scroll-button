## Usage

### Use CDN

```html
<wc-scroll-button/>
<script src="https://cdn.jsdelivr.net/npm/wc-scroll-button@latest" type="module"></script>
```

### Use NPM

```bash
# install
npm i wc-scroll-button
```
```js
// import
import "wc-scroll-button";
```
```html
<!-- use -->
<wc-scroll-button/>
```
