## Setup（CDN or NPM）

#### CDN

```html
<script src="https://cdn.jsdelivr.net/npm/wc-scroll-button@latest" type="module"></script>
```

#### NPM

`wc-scroll-button`

## Usage

```html
<wc-scroll-button/>
```
